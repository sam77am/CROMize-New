Quick Questions

Does this work with any Shopify theme?
+
Yes. Cromize works at the data layer — we analyze your Shopify store data through the API, not by inspecting your theme code. The audit works with any Shopify theme, whether it's Dawn, a premium purchased theme, or a fully custom theme. Some fix instructions may reference specific Shopify theme sections that vary by theme, and we flag those clearly so you know what to look for.
My store is new — do I need historical data?
+
The core store audit — which checks UX, mobile experience, trust signals, checkout friction, and page structure — runs immediately and doesn't require historical order data. The AOV engine and fix attribution features improve with more order history. If your store is under 30 days old, we'll flag which features are limited and which are ready to go from day one.
How is my store data kept secure?
+
Your store data is stored encrypted on SOC 2 certified infrastructure. We request read-only Shopify API access scoped to exactly what we need for the audit — we never request write access, and we never touch your theme files or order data beyond reading it. Full details are in our Privacy Policy.
What if I need help implementing a fix?
+
About 80% of fixes are self-serve — no code required. For the rest, we clearly label them "Developer needed" and include enough detail that you can send the instructions directly to any Shopify developer without a lengthy briefing. Growth and Agency plan customers also have chat support for questions about specific fixes.
